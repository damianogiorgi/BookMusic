############################################
# STRUDEL SYNTAX — USE ONLY WHAT IS BELOW  #
############################################

Strudel plays repeating CYCLES. Notes in a string spread evenly across a cycle.
For background music we want SLOW, so use few notes and slow modulation.

## Tempo — keep it LOW for reading
- setcpm(N) on its own first line. Use 20-45 for calm scenes, up to ~70 only for lively ones.

## Notes / harmony (favor sustained chords and slow melodies)
- note("c3 e3 g3")  named notes: letters a-g, optional # or b, octave 2 (low) to 5 (high).
- ~ is a rest.  [ ] groups into one beat.  < > plays ONE item per cycle (good — slow!).
- Chord = comma in brackets: note("<[c3,e3,g3] [a2,c3,e3]>")  one chord per cycle = gentle.
- SAFER melodies: n("<0 2 4>").scale("C:minor") — scale degrees, any numbers sound consonant.

## Sounds — prefer soft synths and mellow instruments
Pads / synths (USE THESE MOST):
  sine, triangle, gm_synth_strings_1, gm_pad_warm, gm_epiano1, sawtooth (soft if low gain + low lpf)
Mellow acoustic (sparingly, for color):
  piano, gm_acoustic_guitar_nylon, gm_flute, gm_acoustic_bass
Drums (RARE, soft, sparse only):
  bd (kick), rim, hh (hi-hat). Keep gain under .35 and patterns minimal, e.g. "bd ~ ~ ~".

## Scales by mood
  calm/warm     -> "C:major"    serious/sad  -> "C:minor"
  dreamy/wonder -> "C:lydian"   mysterious   -> "C:dorian"
  tense/uneasy  -> "C:phrygian"

############################################
# MODULATION — THIS IS HOW YOU VARY THE VIBE
############################################
Signals are continuous waves you feed into effects to make them MOVE slowly.
Available signals: sine, saw, tri, perlin (smooth random), rand (random).
- .range(lo, hi) sets the min/max. .slow(N) makes the movement take N cycles (use BIG numbers like 8-32 so it drifts gently).

Use these to keep static pads alive:
- Breathing filter (the key ambient move):
    .lpf(sine.range(400, 1600).slow(16))   // brightness slowly opens and closes
- Gentle stereo drift:
    .pan(sine.range(0.2, 0.8).slow(12))
- Soft volume swell:
    .gain(perlin.range(0.3, 0.6))
- Slowly wandering note color (use perlin for smooth):
    .lpf(perlin.range(500, 1200))

## Other useful effects (chain after a pattern)
- .room(0.6)   reverb 0-1 — higher = more distant/spacious (great for pads)
- .lpf(800)    low-pass — LOWER = darker/softer (good for background). Pair with .lpq(N) for resonance.
- .delay(0.4)  echo
- .attack(.5).release(1)  slow fade in/out — makes notes soft-edged, not punchy
- .vowel("<a e o>")  formant filter, gives synths a soft "aah/ooh" vocal quality
- .slow(2) / .fast(2)  stretch or compress timing

############################################
# LAYERING: stack(...)                     #
############################################
stack(
  note("<[c3,e3,g3] [a2,c3,e3]>").sound("gm_synth_strings_1").attack(.4).release(1).gain(.5).room(.6).lpf(sine.range(500,1400).slow(16)),
  note("<c2 a1>").sound("sine").gain(.4).lpf(500)
)
