############################################
# STRUDEL SYNTAX                           #
############################################
Strudel plays repeating CYCLES. Items in a string spread evenly across a cycle.

## Tempo — setcpm(N) on its own first line
- With a DRUM BEAT, one cycle = one bar, so keep N LOW: 18-26 (else beats race).
- For DRUMLESS pad/piano pieces (one chord per cycle), use 28-45.

## Notes / harmony
- note("c3 e3 g3")  named notes: a-g, optional # or b, octave 2 (low) to 5 (high).
- ~ is a rest.  [ ] groups into one beat.  < > plays ONE item per cycle (slow).
- Chord = comma in brackets: note("<[c3,e3,g3] [a2,c3,e3]>")  one chord per cycle.
- Scale degrees: n("<0 2 4>").scale("C4:minor") — any numbers sound consonant.

## Sounds
- Pads/synths: sine, triangle, sawtooth, gm_synth_strings_1, gm_pad_warm, gm_epiano1
- Keys/plucked: piano, gm_acoustic_guitar_nylon, gm_electric_guitar_jazz, gm_vibraphone, gm_celesta, gm_music_box
- Mellow winds/voices: gm_flute, gm_clarinet, gm_ocarina, gm_choir_aahs, gm_reed_organ
- Orchestral (epic): gm_string_ensemble_1, gm_brass_section, gm_french_horn, gm_timpani
- Bass: gm_acoustic_bass, or a low sine
- Drums: bd (kick), sd (snare), hh (hi-hat), rim, cp (clap). Optional vinyl hiss: crackle

## Scales by mood
  calm/warm   -> "C:major"     serious/sad -> "C:minor"
  dreamy      -> "C:lydian"     mysterious  -> "C:dorian"
  tense       -> "C:phrygian"

############################################
# DRUMS — gentle, for beat styles only      #
############################################
AMBIENT, DREAMY, WARM PIANO and EPIC stay DRUMLESS. LOFI, CHILLHOP/JAZZY and
HAPPY/POP use a soft beat; ANGER uses an insistent kick:
- Half-time backbeat: sound("bd ~ sd ~")   kick on 1, snare on 3.
- Insistent (anger): sound("bd ~ bd ~")     kick on 1 and 3, no snare.
- Hats: sound("hh*4")  (gentle). Avoid very busy hats like hh*16.
- Keep drums quiet: kick ~.4, snare ~.35, hats ~.15.
- Optional warmer machine sound: .bank("RolandTR808").

############################################
# MODULATION & EFFECTS                     #
############################################
Signals are slow waves you feed into effects so the sound MOVES: sine, perlin (smooth random).
- .lpf(sine.range(400,1600).slow(16))  breathing brightness (the key ambient move)
- .pan(sine.range(0.2,0.8).slow(12))   gentle stereo drift
- .gain(perlin.range(0.3,0.5))         soft volume swell

Useful effects (chain after a pattern):
- .room(0.6)   reverb 0-1 (higher = more spacious)
- .lpf(800)    low-pass, lower = darker/softer; .lpq(N) adds resonance
- .delay(0.3)  echo
- .attack(.5).release(1)  soft fade in/out
- .vowel("<a e o>")  soft vocal quality on synths
